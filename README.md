# GEO-DrawLine
繪製 Gmap 線路圖並轉出

## Install
npm install

## Angular-Cli
npm install -g angular-cli

## Run
npm start

## Step

1. click Draw
2. draw the map
3. click the Finish if ok
4. click the Revert if you want to redraw
5. color picker can change the line-color
6. click Info to see data window
7. click Export to see JSON data
